Boli Help — Pro Enhancements (AI/UX) — FULL
-------------------------------------------
1) Skopiuj folder `assets/` do projektu.
2) Dodaj przed </body> na stronach:
<link rel="stylesheet" href="assets/bh-pro-enhance.css" data-bh-pro-enhance>
<script src="assets/bh-features-config.js"></script>
<script src="assets/bh-init.js"></script>
<script src="assets/bh-bodymap.js"></script>
<script src="assets/bh-patterns.js"></script>
<script src="assets/bh-tracker.js"></script>
3) Odśwież stronę (Ctrl+F5). Sprawdź konsolę: powinno być "BH Pro Enhancements loaded".
